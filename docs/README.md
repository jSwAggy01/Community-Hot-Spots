# Project Documents

This directory contains planning documents, wireframes, and graphical assets used to construct the logic and design of **Community Hot Spots**.

## Contents

* **`PRD1.docx`**: The original Product Requirements Document (PRD) outlining the tech stack, purpose, and scope of the project.
* **`P1 - User Flow Diagram.drawio` & `.png`**: A flowchart detailing how users navigate through the application, specifically when exploring or adding spots.
* **`P1-wireframe.drawio.xml` & `.png`**: Wireframe sketches for the UI design layout, showing where the sidebar, map, search bar, and add controls will be placed.
* **`example-landing-page.jpg`**: An inspirational reference image used to formulate the styling and aesthetic of the app.
* **`demo.gif`**: A short screen recording showcasing the live mapping functionality in action.
* **`README.md`**: Guide to the files within this directory.
